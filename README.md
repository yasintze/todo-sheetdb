# Simple Todo app bootstrapped with Next.js and SheetDB API

Live Preview : https://todo-sheetdb.netlify.app/

Screenshot Preview

## Access Locally

```
$ git clone https://github.com/yasintze/todo-sheetdb.git
$ cd todo-sheetdb
$ yarn install
$ yarn dev
```
